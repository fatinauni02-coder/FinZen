# 💰 FinZen — Personal Financial Planning System

FinZen is a full-stack personal finance web application that helps young adults manage their money in one place. Users can track their monthly budget, set savings goals, explore financial tips, and manage their personal profile — all through a clean and easy-to-use interface.

---

## 🌟 Features

- **Budget Management** — Set monthly spending limits by category and log daily expenses
- **Savings Goals** — Create and track savings goals with target amounts and deadlines
- **Financial Tips** — Browse, filter, save favourite tips, and submit your own
- **Profile Management** — View and update personal info, change password, or delete account
- **Admin Dashboard** — Manage user accounts and moderate community-submitted tips
- **Responsive Design** — Works on both desktop and mobile devices

---

## 🛠️ Technologies Used

**Frontend**
- HTML5, CSS3, JavaScript
- Bootstrap 5
- Bootstrap Icons
- Google Fonts (DM Sans, DM Serif Display)

**Backend**
- Node.js
- Express.js
- JSON (data storage)

---

## 🚀 How to Run the Project

1. Make sure you have **Node.js** installed on your machine
   - Download from [nodejs.org](https://nodejs.org)

2. Clone this repository
```bash
   git clone https://github.com/yourusername/finzen.git
```

3. Navigate into the project folder
```bash
   cd finzen
```

4. Install dependencies
```bash
   npm install
```

5. Start the server
```bash
   node app.js
```

6. Open your browser and go to
http://localhost:3000
